# Express Book Reviews Application

Este es el proyecto final del curso "Server-side Development with NodeJS, Express and MongoDB". Se trata de una aplicación de catálogo de libros que permite a los usuarios buscar información, registrarse, iniciar sesión y gestionar reseñas.

## Características Principales

* **Gestión de Usuarios:** Registro e Inicio de sesión mediante JSON Web Tokens (JWT).
* **Búsqueda de Libros:** Consultas por ISBN, Autor y Título implementadas de forma asíncrona mediante **Promesas**.
* **Reseñas:** Los usuarios autenticados pueden añadir, modificar y eliminar sus propias reseñas.
* **Arquitectura:** Construido con Node.js y el framework Express.

## Tecnologías Utilizadas

* Node.js
* Express.js
* JSON Web Token (JWT)
* Express-session
* JavaScript (ES6+)

## Requisitos para Ejecución Local

1. Clonar el repositorio.
2. Ejecutar `npm install` para instalar las dependencias.
3. Iniciar el servidor con `npm start`.
4. El servidor correrá en `http://localhost:5000`.
